# Titanic Survival Prediction and Analytics Dashboard

## Project Overview

This project uses Machine Learning techniques to predict Titanic passenger survival and visualize insights using Power BI.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Power BI

## Features

* Data Cleaning
* Exploratory Data Analysis
* Survival Prediction Model
* Dashboard Visualization

## Machine Learning Model

Random Forest Classifier was used for prediction.

## Dashboard Insights

* Survival Count Analysis
* Gender Survival Comparison
* Passenger Class Insights
* Fare Analysis
